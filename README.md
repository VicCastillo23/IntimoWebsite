# Íntimo Coffee — sitio estático

Landing informativa de una sola página (HTML + CSS + JS mínimo). Estética inspirada en sitios de cafeterías premium: tipografía Fraunces + DM Sans, paleta café / verde y hero con imagen de fondo.

## Uso local

Abre `index.html` en el navegador o sirve la carpeta con cualquier servidor estático, por ejemplo:

```bash
cd IntimoCafeSite && python3 -m http.server 8080
```

Luego visita `http://localhost:8080`.

## Personalizar

- Textos y enlaces en `index.html`.
- Colores y layout en `css/styles.css` (variables `:root`).
- Imagen del hero: URL en `.hero-bg` dentro de `styles.css`.
